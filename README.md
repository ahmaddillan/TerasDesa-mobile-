Ikuti langkah-langkah berikut untuk menginisiasi dan menjalankan proyek teras_desa di perangkat lokal Anda:

1. Klon Repositori
Klon repositori ini ke mesin lokal Anda menggunakan perintah Git:

2. Mengambil Dependensi (Wajib!)
Setelah kloning, Anda wajib menjalankan perintah berikut di dalam direktori proyek untuk mengambil semua paket dan dependensi yang diperlukan oleh Flutter. Ini akan memperbaiki error yang mungkin muncul setelah klon.

3. Menjalankan Aplikasi
Setelah dependensi berhasil diunduh, Anda dapat menjalankan aplikasi di emulator atau perangkat fisik Anda:

flutter run
